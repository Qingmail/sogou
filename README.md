# 搜狗输入法 for Mac 皮肤

圆角：名字后面的数字代表圆角大小，`10` 像素圆角小， `13` 像素圆角大

修改：背景在 Photoshop 或其他软件制作好，然后打开[搜狗皮肤编辑器](https://pinyin.sogou.com/mac/softdown.php?r=skineditor)进行细节调整。

> 注意：普通屏幕显示效果圆角会变大，建议在 Retina 屏幕下使用。
